# 2D Football Game

A 2D football game engine built using Python and Pygame. This project demonstrates realistic player movement, ball physics, and a mini football match loop.

## Features
- Smooth player movement with acceleration & friction
- Ball physics and collision system
- Kicking mechanics
- Multi-player support (2 players)
- Foundation for AI and world cup mode
